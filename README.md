## 🛠️ สิ่งที่สคริปต์ของเราเข้าไปปรับแก้ (Technical Features)
*นี่คือรายการสิ่งที่เราปรับให้ลูกค้าแบบเจาะลึกภายในเวลาไม่ถึง 1 นาที:*

1. **Telemetry & Diagnostics Disable:** ปิดระบบสอดแนมและส่งข้อมูลกลับของ Microsoft ที่แอบกิน CPU และเน็ตคุณตลอดเวลา
2. **Xbox Game Bar & DVR Disable:** ปิดระบบอัดจอเบื้องหลังของ Windows ที่เป็นตัวการหลักทำให้เกมหน่วง
3. **Background Apps Suspension:** ปิดการทำงานของแอปพลิเคชันเบื้องหลังที่ไม่ได้ใช้ เพื่อคืนพื้นที่ RAM ให้ FiveM ดึงแมพได้เร็วขึ้น
4. **Ultimate Performance Power Plan:** ปลดล็อกและบังคับเปิดโหมดการจ่ายไฟระดับสูงสุด เพื่อให้ CPU ดันคล็อก (Clock Speed) นิ่งตลอดการรันเกม
5. **Priority Control & System Responsiveness:** ปรับแต่งระบบคิวของคอร์ CPU ให้ความสำคัญและจัดลำดับ Application แบบ Fullscreen (FiveM) มาเป็นอันดับ 1 เสมอ
6. **Network Throttling Removal:** ปลดล็อกข้อจำกัดแบนด์วิดท์เครือข่ายของ Windows มัลติมีเดีย (ทำให้ Sync ผู้เล่นในเซิร์ฟเวอร์ได้เสถียรขึ้น)
7. **Windows Defender Cache Exclusion:** ยกเว้นการสแกนโฟลเดอร์ Cache ของ FiveM แบบเรียลไทม์ ทำให้ฮาร์ดดิสก์/SSD ไม่ต้องทำงานหนักซ้ำซ้อนเวลาขับรถเข้าโซนใหม่ๆ
8. **Disable Fullscreen Optimizations & Visual Effects:** ปิดระบบตีขอบหน้าต่างปลอมๆ ของ Windows 10/11 ให้เกมดึงพลังการ์ดจอ (GPU) ออกมาได้เพียวๆ
9. **Advanced QoS Network Policy (FiveMLag Fix):** จัดลำดับความสำคัญของแพ็กเก็ตอินเทอร์เน็ตให้ไฟล์ `FiveM.exe` ส่งผ่านเราเตอร์ก่อนแอปพลิเคชันอื่นๆ ในเครื่อง

## ⚙️ การตั้งค่าเพิ่มเติมด้วยตัวเอง (Manual Settings สำหรับช่าง)

นอกเหนือจากสคริปต์ Optimizer แล้ว ช่างสามารถตั้งค่า 2 ส่วนนี้เพิ่มเติมด้วยตัวเองเพื่อให้ลูกค้าเล่น FiveM ได้ลื่นไหลที่สุดและดัน FPS ให้ทะลุขีดจำกัดสูงสุด:

### 1. การตั้งค่า \`citizen.fx.ini\` (ในโฟลเดอร์ FiveM Application Data)
เปิดไฟล์ \`citizen.fx.ini\` ด้วย Notepad จากนั้นให้เพิ่มบรรทัดตั้งค่าระดับสูงลงไป:
\`\`\`ini
[Game]
UpdateChannel=production
; บังคับใช้ GPU ตัวหลักในการประมวลผลเสมอ ป้องกันอาการออนบอร์ดแอบแย่งซีน
ConVar.sys_info_override_vram_size=8192 ; (ใส่เลข RAM ของการ์ดจอเป็น MB เช่น การ์ดจอ 8GB ใส่ 8192, 12GB ใส่ 12288)
[Video]
; ล็อคคิวงานกราฟิกให้การ์ดจอแยกเท่านั้น
DedicatedVideoMemory=8192 ; (ใส่เท่ากับด้านบน)
\`\`\`

### 2. การตั้งค่า NVIDIA Control Panel (คลิกขวาหน้าเดสก์ท็อป)
สำหรับลูกค้าที่ใช้การ์ดจอค่ายเขียว แนะนำให้เข้าไปที่ **Manage 3D Settings** -> **Program Settings** -> และเลือกเพิ่มไฟล์ \`FiveM.exe\` และ \`GTA5.exe\`:
- **Power management mode:** \`Prefer maximum performance\` *(สำคัญมาก ทำให้ไฟเลี้ยงการ์ดจอไม่ดรอปตอนจังหวะบวก)*
- **Texture filtering - Quality:** \`High performance\`
- **Low Latency Mode:** \`Ultra\` หรือ \`On\` *(เคล็ดลับสำหรับสายรำ ทำให้เมาส์ติดหน้าจอไม่ดีเลย์)*
- **Max Frame Rate:** ปิด (\`Off\`) *(แนะนำให้ไปจำกัด FPS ใน Setting บนเซิร์ฟเวอร์แทนการล็อคที่ไดร์เวอร์)*
- **Threaded optimization:** \`On\`
- **Vertical sync:** \`Off\`

*เมื่อทำครบทั้งรันสคริปต์ \`FiveM_Optimizer_V2.ps1\` ควบคู่กับการตั้งค่า Manual ตรงนี้ รับประกันว่าเครื่องลูกค้าจะตอบสนองเมาส์ได้ไวที่สุดแน่นอน 100%!*
