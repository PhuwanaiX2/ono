# 📖 คู่มือสำหรับช่าง (Technician Guidelines)
**ข้อควรระวังและสคริปต์การคุยกับลูกค้าก่อนกดปรับแต่ง (Optimize) เครื่อง**

เอกสารนี้มีไว้เพื่อให้ช่าง/แอดมินใช้เตรียมความพร้อมลูกค้าก่อนรัน `FiveM_Optimizer_V2.ps1` โดยเฉพาะลูกค้าที่ซีเรียสเรื่อง FPS, ความสมูทของภาพ (ตัวรำ) และต้องการให้มูฟเมนต์การยิงตอบสนองไวที่สุด (Input Lag ต่ำสุด)

---

## 🛑 1. สิ่งที่ต้องแจ้งลูกค้าให้ทราบ "ก่อนทำ" (สำคัญมาก)

เพื่อลดปัญหาลูกค้าทักมาโวยวายทีหลัง ช่างควรบรีฟและทำความเข้าใจกับลูกค้าในประเด็นเหล่านี้:

- **🔐 ด้านความปลอดภัย (Security):**
  > **คำพูดแนะนำ:** *"พี่ครับ สคริปต์นี้จะรีด FPS ให้สุดทางภาพสมูทขึ้นแน่นอน แต่มันจะไปปิดระบบป้องกันไวรัสระดับลึกของ Windows (Core Isolation) และยกเว้นการสแกนโฟลเดอร์เกม FiveM เพื่อไม่ให้เกมกระตุก ดังนั้นถ้าพี่ลงของเถื่อนหรือโหลดโปรแกรมแปลกๆ นอกเหนือจากเกม พี่ต้องระวังตัวนิดนึงนะครับ เครื่องมันจะติดไวรัสได้ง่ายขึ้นนิดหน่อย"*
- **🔎 ระบบค้นหาของ Windows (Windows Search):**
  > **คำพูดแนะนำ:** *"สคริปต์นี้ปิด Service พื้นหลังไปเยอะมากเพื่อคืน RAM ให้เกม รวมไปถึงระบบ Search ด้วย ถ้าพี่กดปุ่ม Start แล้วพิมพ์หาแอปหรือไฟล์ไม่เจอ ไม่ต้องตกใจนะครับ มันคือผลจากการปิด Service เพื่อให้เกมลื่นที่สุด"*
- **🔋 สำหรับลูกค้าที่ใช้ โน้ตบุ๊ก (Laptop):**
  > **คำพูดแนะนำ:** *"ถ้าพี่เล่นบนโน้ตบุ๊ก หลังจากทำไปแล้ว ตอนไม่ได้เสียบสายชาร์จแบตจะลดฮวบเลยนะครับ และเครื่องจะร้อนกว่าปกติ เพราะเราสั่งปลดลิมิตจ่ายไฟให้ CPU/GPU วิ่งเต็มตลอดเวลา"*

---

## ✅ 2. อธิบายให้ลูกค้าฟังว่า "ทำไปแล้วได้อะไร?" (Selling Points)

หากลูกค้าถามว่ากด V2 แล้วต่างจากของเดิมหรือทำเองยังไง ให้อธิบายตามนี้:
1. **เมาส์คมขึ้น (1:1 Raw Input):** เราลบอัตราเร่งเมาส์ของ Windows ทิ้ง และบังคับไม่ให้พอร์ต USB หลับ ทำให้หันหน้า สะบัดเมาส์ ยิงแม่นขึ้นแน่นอน
2. **Ping นิ่ง (TCP/IP & QoS):** เราลบดีเลย์การส่งแพ็กเก็ตอินเทอร์เน็ตทิ้ง (เหมือนทำ Leatrix) และจัดคิวให้เน็ตวิ่งเข้า FiveM เป็นอันดับ 1 ทำให้ตัวละครไม่ค่อยวาร์ป หรือยิงแล้วดาเมจขึ้นไวขึ้น
3. **ลดอาการเกมหน่วงหลังแครชบ่อย (Disable FTH):** ปกติถ้า FiveM เด้งบ่อย Windows จะแอบดรอปความเร็วเกมลง (FTH) เราปิดทิ้งให้หมด ทำให้ FPS วิ่งเต็มตลอด
4. **ปลดล็อกคอขวด CPU (VBS/Core Isolation):** เราปิดตัวกินสเปคของ Windows 11 ที่ทำให้ FPS ตกออกไป ทำให้ CPU ทำงานให้เกมได้แบบ 100% เฟรมพุ่งขึ้นแน่นอน

## 🛠️ 3. ข้อบ่งใช้สำหรับช่าง (Technical Notes)

*   **ลูกค้าใช้ HDD (ฮาร์ดดิสก์จานหมุน):** ถ้าลูกค้าไม่ได้ใช้ SSD การปิด `SysMain` (Superfetch) อาจจะทำให้เค้าเข้าเกมช้ากว่ารอบแรกๆ ให้ช่างรู้ไว้เผื่อลูกค้าทักมาถามว่าทำไมโหลดแมพนานขึ้น
*   **เมื่อลูกค้าอยากยกเลิก:** ให้รันไฟล์ `FiveM_Restore_V2.ps1` หรือ ย้อน System Restore Point ที่สคริปต์สร้างไว้ตอนแรก
